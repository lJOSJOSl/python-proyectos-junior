# Proyectos en Python

Este repositorio contiene varios proyectos desarrollados para practicar programación en Python.
Cada proyecto implementa conceptos fundamentales como estructuras de datos, manejo de archivos y lógica de negocio.

## Proyectos incluidos 📂

### Sistema de Biblioteca 📚
Aplicacion que permite gestionar libros y usuarios dentro de una biblioteca.

Funciones:
- Agregar libros
- Registrar usuarios
- Prestar libros
- Ver disponibilidad

Archivo principal:
Biblioteca.py

---------------------------------------

### Sistema de Caja 🧾
Simulacion de un sistema de ventas o caja registradora.

Funciones:
- Agregar productos
- Calcular total
- Generar ticket
- Guardar ventas

Archivo principal:
main.py

---------------------------------------

### CRUD de empleados 👨‍💼
Sistema para administrar la información de empleados.

Funciones:
- Crear empleado
- Ver empleados
- Actualizar datos
- Eliminar empleados

---------------------------------------

### Tecnologías utilizadas 🛠

- Python
- Archivos JSON
- Programación modular

---------------------------------------

### Objetivos 🎯

Practicar conceptos básicos de programación y foñrtalecer habilidades en desarrollo con python.


## Cómo ejecutar los programas

1. Clonar el repositorio
2. Ejecutar el archivo principal

Ejemplo:

Python Sistema_caja/main.py

En los readme encontraras el archivo principal si no identificas cual es.
